## 🌟 Overview

**Tech Navyug** is a high-performance, modern web application designed for a seamless user experience. Built with the latest **React 19** and **Vite**, it combines speed, efficiency, and a premium aesthetic to provide a top-tier platform for both users and administrators.

Whether you're browsing through innovative products, managing an inventory, or engaging with rich technical content, Tech Navyug provides a robust and scalable architecture to handle it all.

---

## ✨ Features

- **💎 Premium UI/UX**: Built with a focus on modern aesthetics, gradients, and smooth animations using Tailwind CSS 4 and Lucide icons.
- **⚡ Super Fast Rendering**: Leverages React 19 and Vite 8 for near-instant boot times and HMR.
- **🛡️ Secure Authentication**: Role-based access control (RBAC) ensuring only authorized users access the admin dashboard.
- **🛒 Dynamic Marketplace**: A fully functional product listing with advanced filtering, categorization, and a Redux-powered cart system.
- **📊 Admin Dashboard**: Comprehensive management tools for products, services, and content.
- **✍️ Rich Text Editing**: Integrated Tiptap editor for creating and managing high-quality markdown and technical content.
- **📱 Fully Responsive**: Optimized for every screen size, from mobile phones to high-resolution desktop monitors.
- **🌓 Dark Mode Ready**: Deeply integrated dark/light theme support for all components.

## 🛠️ Tech Stack

### Frontend Core
- **Framework**: [React 19](https://react.dev/) (The latest and greatest)
- **Build Tool**: [Vite 8 (Beta)](https://vite.dev/) (Speed redefined)
- **Routing**: React Router 7

### State & Data
- **State Management**: Redux Toolkit (Slices & API Middleware)
- **Data Fetching**: TanStack Query (React Query) for efficient caching and synchronization.
- **API Client**: Axios with interceptors for auth handling.

### Styling & UI
- **Styling**: Tailwind CSS 4 (The future of utility-first CSS)
- **Components**: Shadcn/UI (Radix UI) for accessible, unstyled components.
- **Icons**: Lucide React & React Icons.
- **Animations**: Tailwind Animate & Framer Motion.

### Form & Validation
- **Forms**: React Hook Form.
- **Validation**: Yup Schema validation.

## 📂 Project Structure

```bash
src/
├── api/          # API definitions and Axios instances
├── components/   # Reusable UI components (Layout, Common, UI)
├── context/      # React Contexts for global state
├── hooks/        # Custom React hooks
├── pages/        # Main application pages (Public & Admin)
├── routes/       # Centralized routing configuration
├── services/     # Business logic and external service integrations
├── store/        # Redux Toolkit configuration and slices
├── styles/       # Global styles and Tailwind configurations
└── utils/        # Helper functions and constants
```

## 🚀 Getting Started

### Prerequisites

- **Node.js**: v18.x or higher
- **npm**: v9.x or higher

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/technavyug-frontend.git
   cd technavyug-frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your API endpoints:
   ```env
   VITE_API_BASE_URL=http://localhost:5000/api
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
